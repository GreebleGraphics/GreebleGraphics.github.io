The Head -- Weight Maps

STEP 1: Load head13.lwo into Modeler (if you just want to practice weight mapping, you may load head13.lwo from the CD-ROM).

STEP 2: Set the Perspective View window"s display type to Weight Shade.

STEP 3: Select the SubPatch surfaces of the ponytail underneath the upper ponytail holder, that do not touch the upper ponytail holder. Create a new weight map called "Ponytail," with initial weight map values set to 100%. (Figure 1)
 

Figure 1
Figure 2

STEP 4: Type the double-quotes sign to Invert your selection. (Display > Selection > Sel Invert). Create a new weight map for the selected geometry, called "Head and Bangs," with initial weight map values set to 100%. (Figure 2)

STEP 5: Save this object as head14.lwo.
